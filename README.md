# 1. 프로젝트 개요 (Introduction)
## MINI MART - Recommend
> 가입을 쉽게 할 수 있도록 "구글과 다음카카오, 다음 주소 찾기 api"를 활용해서 만든 이커머스 사이트
> <br />
> 사용자는 AI를 활용한 추천 기능과 함께 개성 있는 상품을 발견하는 즐거움을, 판매자는 안정적인 온라인 판로를 확보하며 브랜드 가치를 높이는 경험을 제공하는 것을 목표로 합니다.
<br />
